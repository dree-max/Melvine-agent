# Daily Briefing Command

Use this command to get a comprehensive daily briefing with schedule, tasks, and reminders.

## Usage

`/daily-briefing` or `/briefing`

## What to Do

Provide a structured daily briefing with the following sections:

```
☀️ GOOD [MORNING/AFTERNOON/EVENING]!

📅 TODAY'S SCHEDULE
• [Time] - [Event 1]
• [Time] - [Event 2]
• (list all events in chronological order)

✓ PRIORITIES TODAY
• [High] [Task 1]
• [Medium] [Task 2]
• [Low] [Task 3]
• (list tasks by priority)

🔔 REMINDERS
• [Time] - [Reminder 1]
• [Time] - [Reminder 2]
• (list all reminders)

💡 PRODUCTIVITY TIP
• [One helpful suggestion based on their schedule/tasks]

📝 NOTES
• [Any relevant context from previous conversations]
```

## Context

- Use appropriate greeting based on time of day (Uganda EAT timezone)
- Reference any ongoing projects or follow-ups from previous conversations
- Consider the day's schedule density when giving productivity tips
- If schedule is packed, suggest time-blocking strategies
- If schedule is light, suggest focusing on high-priority tasks

## Additional Options

- `/briefing week` - Show weekly overview
- `/briefing tomorrow` - Show tomorrow's schedule
- `/briefing tasks` - Show only tasks without schedule
